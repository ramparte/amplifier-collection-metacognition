# Programmatic Usage Examples

This directory contains code examples showing how to integrate and use the metacognition collection programmatically.

## Examples

- **basic_usage.py**: Simple example using complexity-assessor
- **iterative_workflow.py**: Complete iterative refinement workflow
- **ensemble_example.py**: Ensemble coordination example
- **error_handling.py**: Error handling and recovery patterns
- **custom_integration.py**: Integrating with existing Amplifier workflows

## Note

These examples are illustrative and show the **conceptual usage patterns**. The actual Amplifier API may differ - refer to official Amplifier documentation for current API.

The examples demonstrate:
- How to structure inputs for agents
- How to parse agent outputs
- How to chain agent calls
- How to handle errors gracefully
